VOICE TO TEXT CHATTER

Overview

Communication has become an essential aspect of modern life, but typing long messages or emails can be challenging for some people. To address this issue, the Voice to text chatter plugin allows users to convert their spoken words into text.

Who is this Plugin for?

This plugin is primarily designed for existing Odoo users who engage in conversations using the Odoo Discuss app. It is also perfect for people who want to bypass the inconvenience of text-based functions and prefer to use their voice for communication. Also, this plugin is ideal for those who find it challenging to type messages due to reasons such as a lack of typing skills or disabilities.

Addressed issue

The main problem with traditional messaging systems is that they are unable to translate the information to satisfy the chatters' intent. Moreover, typographical errors are common, leading to incorrect messages. This can cause misunderstandings, delays, and frustration among users.

Solution

The Voice-to-text chatter plugin provides a convenient solution for users who prefer to communicate using their voice. With its voice-to-text feature and use of Google web speech API, the plugin delivers accurate and relevant chat messages, creating quicker messages and enhancing communication speed. The user-friendly design makes it easy to use, even on mobile phones, allowing for communication on the go.

Working

To initiate communication on your Odoo store using voice, follow these steps:

1. Install the Voice Speech module.

2. Click on the Microphone button located in the message box.

3. Speak the message you want to convey.

4. Press the Enter key to send the message.
